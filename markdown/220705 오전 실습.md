### Fenced Code Block

- `(backtick) 기호 3개를 활용하여 작성한다.
- 특정 언얼르 명시하면 Syntax highlighting 기능이 적용된다.

```python
print('helloworld')

```

```python
print('hello')
#주석

if True:
  print('t')
  
  else:
    print('f')
```



```html
print('hello')
# 주석?
<h1>
  제목
</h1>
<!-- 주석 -->
```
