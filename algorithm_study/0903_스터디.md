#### BOJ 1697 - 🔍 숨바꼭질

##### 1. **필요 개념 : BFS, 1차원 배열, queue**



##### 2. **선택 가능한 방향 : 총 3가지**

➡  `X-1`, `X+1`, `2*X`



##### 3. **queue에 넣게 되는 경우 고려사항**

```bash
1. 시간 정보 (몇 초가 흘렀는지) 가 들어가야 함

2. 몇 초가 흘렀는지 기록되어 있어야 함

3. 중복 방문을 하지 않음
```
