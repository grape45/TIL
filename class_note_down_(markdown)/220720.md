#### 클래스

- 클래스 속성(attribute)

  - 한 클래스의 모든 인스턴스라도 똑같은 값을 가지고 있는 속성
  - 클래스 선언 내부에서 정의
  - <classname>.<name>으로 접근 및 할당

- 인스턴스와 클래스 간의 이름 공간(namespace)

  - 클래스를 정의하면, 클래스와 해당하는 이름 공간 생성
  - 인스턴스를 만들면, 인스턴스 객체가 생성도고 이름 공간 생성
  - 인스턴스에서 특정 속성에 접근하면, 인스턴스 - 클래스 순으로 탐색

- 클래스 메소드
