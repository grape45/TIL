#### 1. 순회

1. 이중 for문을 이용한 **행 우선 순회**

   ```python
   matrix = [
     [1, 2, 3, 4],
     [5, 6, 7, 8],
     [9, 0, 1, 2]
   ]
   
   for i in range(3): # 행 THEGOD
     for j in range(4): # 열
       print(matrix[i][j], end = " ")
     print()
   # 1 2 3 4 
   # 5 6 7 8
   # 9 0 1 2
   ```

   > 이해를 돕기 위한 표

   | 행(i) | 열(j) | matrix`[i][j]` |
   | ----- | ----- | :------------- |
   | 0     | 0     | 1              |
   | 0     | 1     | 2              |
   | 0     | 2     | 3              |
   | 0     | 3     | 4              |
   | 1     | 0     | 5              |
   | 1     | 1     | 6              |
   | 1     | 2     | 7              |
   | 1     | 3     | 8              |
   | 2     | 0     | 9              |
   | 2     | 1     | 0              |
   | 2     | 2     | 1              |
   | 2     | 3     | 2              |

2. 이중 for문을 이용한 열 우선 순회

   ```python
   matrix = [
     [1, 2, 3, 4],
     [5, 6, 7, 8],
     [9, 0, 1, 2]
   ]
   
   for i in range(4): # 열
     for j in range(3): # 행
       print(matrix[j][i], end = " ")
     print()
   # 1 5 9
   # 2 6 0
   # 3 7 1
   # 4 8 2
   ```

   > 이해를 돕기 위한 표

   | 열(i) | 행(j) | matrix`[j][i]` |

   | 1     | 0     | 2              |
   | 1     | 1     | 6              |

   | 2     | 1     | 7              |
   | 2     | 2     | 1              |
   | 3     | 0     | 4              |
