1. 이차원 리스트

   - 리스트를 원소로 가지는 리스트일 뿐
   - 이차원 리스트 == **행렬(matrix)**

   ```PYTHON
   #          0  1  2    0  1  2    0  1  2
   matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
   #             0          1          2
   
   # 보기좋게 변경하면 행렬(matrix)의 형태가 됨
   
   matrix = [
    0 [1, 2, 3],
    1 [4, 5, 6],
    2 [7, 8, 9]
       0  1  2
   ]
   
   print(matrix[0][0]) # 1
   print(matrix[1][2]) # 6
   print(matrix[2][0]) # 7
   ```
